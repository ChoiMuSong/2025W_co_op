# 2026W Co-op - Computer Security and Network Practice

This repository is for self-practice and learning about computer security and network topics.

## Purpose

This repository serves as a personal learning space for:
- Computer Security concepts and practices
- Network fundamentals and protocols
- Security best practices
- Presentation materials and study notes

## Structure

```
.
├── presentations/     # PDF presentations and study materials
└── README.md         # This file
```

## Presentations

All presentation materials (PDF files) are organized in the `presentations/` directory. Each presentation covers specific topics related to computer security and networking.

For details on how to organize and contribute presentations, see [presentations/README.md](presentations/README.md).

## Usage

This is a learning repository. Feel free to:
1. Browse presentations in the `presentations/` directory
2. Add new presentation materials
3. Organize topics by subdirectories as needed

## Contributing

To add a new presentation:
1. Place your PDF file in the `presentations/` directory
2. Use descriptive filenames (e.g., `network-security-basics.pdf`)
3. Organize by topic subdirectories if needed
4. Update the presentations README with a brief description
