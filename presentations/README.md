### Naming Convention

Use descriptive filenames that clearly indicate the topic:
- Good: `intro-to-network-security.pdf`
- Good: `buffer-overflow-attacks.pdf`
- Good: `tcp-ip-protocol-overview.pdf`
- Avoid: `presentation1.pdf`, `slides.pdf`
